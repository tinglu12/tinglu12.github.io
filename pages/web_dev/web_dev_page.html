Open Course

Work in progress

Open Course is a one stop shop for Michigan students to find a study group and other course resources that are perfect for them. Users will be able to search for their class and see related shared google drives, chats, and posts for study groups. The time periods for the study groups will be posted in calendar form, so users can compare it to their personal calendar and see what works best for them.

VirtuWear

VirtuWear is a hackathon project that offers personalized outfit recommendations by utilizing user prompts and previous clothing images. Developed with Flask for the backend and React for the frontend, alongside SQLite for the database, we overcame challenges in image processing and feature integration. Notably, our AI accurately categorizes user images for relevant suggestions. This project taught us valuable lessons in web app development and leveraging UM-GPT AI. Future plans involve refining the UI and expanding the AI's database for improved recommendations.
