Open Course

Open course is a project that is currently being developed by Void Tech at University of Michigan. With Open Course, students will be able to join study groups and obtain resources for whatever courses they are taking at UMich. At Void Tech, we are dedicated towards providing the resources you need to conquer that exam and major as a student! Currently, this project is still a work in progress, but we hope to release it by Fall 2024!

For Frontend, we are currently using React to create reusable components. For Backend, we are using Flask to create our APIs. Our Database is being hosted on AWS and using PostgreSQL. The overall design of our project was created using Figma.
